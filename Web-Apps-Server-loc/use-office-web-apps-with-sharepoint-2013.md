# Utiliser Office Web Apps avec SharePoint 2013

 

_<strong>Sapplique à :</strong>Office Web Apps, SharePoint Foundation 2013, SharePoint Server 2013_

_<strong>Dernière rubrique modifiée :</strong>2017-05-26_

**Résumé :** contient des articles qui vous permettront de découvrir comment déployer et configurer Office Web Apps pour une installation locale de SharePoint 2013.

**Public concerné** : professionnels de l’informatique

Conjugué à une installation locale de SharePoint 2013, Office Web Apps fournit des versions mises à jour des applications Word Web App, Excel Web App, PowerPoint Web App et OneNote Web App. Les utilisateurs peuvent consulter, et éventuellement modifier, des documents Office s’ils utilisent un navigateur web reconnu sur les ordinateurs et divers appareils mobiles (Windows Phone, iPhone, iPad, etc.). Outre les nouvelles fonctionnalités d’Office Web Apps, l’architecture et les méthodes de déploiement ont également été modifiées, ce qui permet à Office Web Apps de fournir des fonctionnalités d’affichage pour Lync Server 2013.
