# Déployer l'infrastructure : Office Web Apps Server

 

_**Sapplique à :** Office Web Apps Server_

_**Dernière rubrique modifiée :** 2016-12-16_

**Résumé :** contient des articles relatifs à Office Web Apps Server et la manière dont il distribue la fonctionnalité Office Web Apps aux hôtes.

**Public concerné** : professionnels de l'informatique

Office Web Apps Server est un nouveau produit Office Server qui offre des versions de Word, PowerPoint, Excel et OneNotebasées sur le navigateur. Une seule batterie Office Web Apps Server peut prendre en charge des utilisateurs qui accèdent aux fichiers Office via SharePoint 2013, Lync Server 2013, Exchange Server 2013, des dossiers partagés et des sites web. Le nouveau modèle de déploiement autonome signifie que vous pouvez gérer des mises à jour de votre batterie Office Web Apps Server indépendamment des autres produits Office Server déployés dans votre organisation.
